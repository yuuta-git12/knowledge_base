# iOSでのみWebサイトの表示がおかしい場合の対処
## 説明
- iOSでのみWebサイトの表示がおかしい場合に調べたサイトのリンクを記載する
- 3の方法はiPhone実機とMacを使ったデバッグ方法、iOS向けのサイト開発には役立つ

## 実例
- 案件でiOS時のみ表示がおかしくなる事象が発生したが、CSSのpositionの内容を修正することで対応できた。4のサイトがヒントになった
